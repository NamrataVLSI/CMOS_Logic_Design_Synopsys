# CMOS Logic Design using Synopsys Custom Compiler

This repository documents my learning and revision of CMOS logic gate design using Synopsys Custom Compiler and PrimeWave with the SAED 32/28nm PDK.

## Repository Structure

Each logic gate is organized into:
- schematic and pre-layout simulation
- layout documentation

## Gates

- Inverter
- NAND
- NOR
- AND
- OR
- XOR
- XNOR
